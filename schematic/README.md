# 原理图连接说明

## SF32LB52 + ADAU1977 + 6 路麦克风

---

## 1. 网络标签命名规范

| 网络 | 命名 | 说明 |
|------|------|------|
| I2S 时钟 | I2S_BCLK, I2S_LRCLK, I2S_MCLK | 位时钟、左右时钟、主时钟 |
| I2S 数据 | I2S_DIN | 数据输入 (SF32 接收) |
| 电源 | 3V3_DIG, 3V3_ANA | 数字电源、模拟电源 |
| 地 | GND_DIG, GND_ANA | 数字地、模拟地 |
| 麦克风 | MIC1_OUT ~ MIC6_OUT | 6 路麦克风输出 |

---

## 2. SF32LB52 连接

### 2.1 电源引脚

| 引脚 | 网络 | 说明 |
|------|------|------|
| VDD_IO | 3V3_DIG | IO 电源 |
| VDD_CORE | 1.2V (内部 LDO) | 核心电源 |
| VSS | GND | 地 |

**去耦电容**:
- VDD_IO → 100nF → GND (靠近引脚)
- VDD_CORE → 1uF → GND (靠近引脚)

### 2.2 I2S1 接口

| SF32 引脚 | 功能 | 连接 |
|----------|------|------|
| PA24 | I2S1_MCLK | → ADAU1977 MCLK |
| PA25 | I2S1_SDO | (未用，可输出) |
| PA28 | I2S1_SDI | ← ADAU1977 SDATA_OUT |
| PA29 | I2S1_BCK | ← ADAU1977 BCLK |
| PA30 | I2S1_LRCK | ← ADAU1977 LRCLK |

### 2.3 内置 Codec

| SF32 引脚 | 功能 | 连接 |
|----------|------|------|
| PA4 | MIC_ADC | ← 备用模拟麦克风 (可选) |
| PA5 | MIC_BIAS | → 麦克风偏置电压 (3V) |
| PA2 | DAC_P | → 功放 IN+ |
| PA3 | DAC_N | → 功放 IN- |

### 2.4 调试接口

| SF32 引脚 | 功能 | 连接 |
|----------|------|------|
| PA18 | UART0_RXD | ← USB-C (CH340 TX) |
| PA19 | UART0_TXD | → USB-C (CH340 RX) |
| PA18 | SWDIO | → 调试头 (可选) |
| PA19 | SWCLK | → 调试头 (可选) |

### 2.5 按键和 LED

| SF32 引脚 | 功能 | 连接 |
|----------|------|------|
| PA33 | KEY2 | → 按键 → GND |
| PA34 | KEY1 | → 按键 → GND |
| PA31 | LED1 | → LED → 3V3 (低电平亮) |
| PA32 | LED2 | → LED → 3V3 (低电平亮) |

## 3. ADAU1977 连接

### 3.1 电源引脚

| 引脚 | 网络 | 电容 |
|------|------|------|
| AVDD1-4 | 3V3_ANA | 每引脚 100nF + 共用 10uF |
| DVDD1-2 | 3V3_DIG | 每引脚 100nF + 共用 10uF |
| AGND1-4 | GND_ANA | - |
| DGND1-2 | GND_DIG | - |

### 3.2 模拟输入 (6 路麦克风)

```
MIC1_OUT ── C1(1uF) ──┬── R1(1k) ── GND
                      │
                     C2(100nF)
                      │
                     GND
                      │
                     └──→ ADAU1977 AIN1

MIC2_OUT ── C3(1uF) ──┬── R2(1k) ── GND
                      │
                     C4(100nF)
                      │
                     GND
                      │
                     └──→ ADAU1977 AIN2

... (重复到 AIN6)
```

### 3.3 数字接口

| ADAU1977 引脚 | 连接 | 说明 |
|--------------|------|------|
| MCLK | ← SF32 PA24 | 主时钟 12.288MHz |
| BCLK | → SF32 PA29 | 位时钟 (输出) |
| LRCLK | → SF32 PA30 | 左右时钟 (输出) |
| SDATA_OUT | → SF32 PA28 | 数据输出 |
| SDATA_IN | (未用) | - |

### 3.4 I2C 控制接口

| ADAU1977 引脚 | 连接 | 说明 |
|--------------|------|------|
| SCL | ← SF32 I2C_SCL | PA37/PA39 (软件配置) |
| SDA | ← SF32 I2C_SDA | PA38/PA40 (软件配置) |
| ADDR0 | GND | I2C 地址选择 |
| ADDR1 | GND | I2C 地址选择 |

### 3.5 复位

| ADAU1977 引脚 | 连接 | 说明 |
|--------------|------|------|
| RST | ← SF32 GPIO | PA41 (可选，或接 RC 复位) |

**RC 复位电路** (如不用 GPIO 控制):
```
3V3 ── R(10k) ──┬── RST
                │
               C(100nF)
                │
               GND
```

## 4. 麦克风电路 (6 路相同)

### 模拟 MEMS 麦克风 (SPW0442LR5H)

```
     ┌─────────────┐
     │   MICROM    │
     │             │
  1 ─┤ VDD         │
  2 ─┤ GND         │
  3 ─┤ OUT         ├──→ MICx_OUT
  4 ─┤ NC          │
     │             │
     └─────────────┘

连接:
VDD  → MIC_BIAS (3V3 通过 1k 电阻)
GND  → GND
OUT  → 耦合电容 → ADAU1977 AINx
NC   → 悬空
```

### 麦克风偏置电路

```
3V3 ── R(1k) ──┬── MIC_BIAS ──→ MIC1 VDD
               │               → MIC2 VDD
               │               → ...
               │               → MIC6 VDD
               │
              C(10uF)
               │
              GND
```

## 5. 电源电路

### 5.1 USB 输入

```
USB-C VBUS ──┬── FUSE(500mA) ──┬── 5V
             │                 │
            C(10uF)           C(10uF)
             │                 │
            GND               GND
```

### 5.2 3.3V LDO (数字)

```
5V ──┬── XC6206 ──┬── 3V3_DIG
     │  SOT-23-3  │
    C(1uF)       C(10uF)
     │            │
    GND          GND
```

### 5.3 3.3V LDO (模拟)

```
5V ──┬── XC6206 ──┬── 3V3_ANA
     │  SOT-23-3  │
    C(1uF)       C(10uF)
     │            │
    GND          GND
```

**注意**: 模拟和数字电源分开，减少噪声干扰

## 7. 立创 EDA 操作提示

### 7.1 创建网络标签
1. 按 `N` 键放置网络标签
2. 命名规范：`I2S_BCLK`, `3V3_DIG`, `MIC1_OUT` 等
3. 相同名称自动连接

### 7.2 放置电源符号
1. 按 `P` 键放置电源
2. 选择 `VCC`, `GND`, `3.3V` 等
3. 全局电源网络自动连接

### 7.3 检查连接
1. 工具 → 设计规则检查 (DRC)
2. 修复所有电气错误
3. 更新 PCB 前确保原理图无误
