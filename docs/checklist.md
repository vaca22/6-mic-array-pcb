# 设计检查清单

## 原理图阶段

### 电源系统
- [ ] 3.3V LDO 输入输出电容已放置 (靠近引脚)
- [ ] 1.8V/1.2V (如有) 稳压电路完整
- [ ] 所有芯片 VDD 引脚有去耦电容 (100nF)
- [ ] 电源网络命名清晰 (3V3, 1V8, GND)

### 麦克风部分
- [ ] 6 个麦克风 VDD 接 3.3V
- [ ] 6 个麦克风 GND 接地
- [ ] L/R 引脚正确配置 (3 个 GND, 3 个 VDD)
- [ ] PDM 数据线连接到 Hub 对应引脚
- [ ] 时钟线连接到 Hub 时钟输出

### 音频 Hub (ADAU1977)
- [ ] AVDD/DVDD 电源连接正确
- [ ] AGND/DGND 接地正确
- [ ] PDM 输入引脚连接 6 路麦克风
- [ ] I2S 输出引脚 (BCLK, LRCLK, DOUT) 连接 MCU
- [ ] MCLK 输入连接晶振或 MCU
- [ ] I2C 控制接口 (SCL, SDA) 连接 MCU (可选)
- [ ] 复位引脚正确处理

### MCU (STM32F407)
- [ ] VDD/VSS 所有引脚连接
- [ ] NRST 复位电路 (10k + 100nF)
- [ ] BOOT0/BOOT1 配置正确
- [ ] 晶振电路 (8MHz HSE)
- [ ] I2S 引脚分配正确
- [ ] SWD 调试接口引出

### 连接器
- [ ] USB-C 电源输入 (如有)
- [ ] 调试接口 (SWD) 引出
- [ ] 音频输出接口 (如有)

---

## PCB 布局阶段

### 元件布局
- [ ] 麦克风按阵列形状均匀分布
- [ ] 麦克风声孔位置准确 (±0.5mm)
- [ ] 麦克风间距 ≥15mm
- [ ] Hub 位于麦克风组中心
- [ ] MCU 靠近 Hub
- [ ] 电源部分远离模拟信号

### 麦克风区域
- [ ] 声孔区域无铜皮 (所有层)
- [ ] 麦克风下方底层挖空
- [ ] 麦克风周围无过孔
- [ ] 麦克风接地良好 (多个过孔到 GND)

### 走线
- [ ] PDM 走线短而直 (<30mm)
- [ ] PDM 6 路走线等长 (误差 ≤5mm)
- [ ] I2S 走线包地处理
- [ ] 时钟线远离模拟信号
- [ ] 电源走线足够宽 (>15mil)
- [ ] 无锐角走线 (45°或圆弧)

### 地平面
- [ ] 完整地平面 (无断裂)
- [ ] 模拟地数字地正确分割 (如有)
- [ ] 过孔阵列连接多地平面
- [ ] 麦克风 GND 多点接地

---

## DRC 检查

### 电气规则
- [ ] 无短路错误
- [ ] 无开路错误
- [ ] 无未连接网络
- [ ] 所有电源网络连通

### 物理规则
- [ ] 线宽 ≥6mil (工厂能力)
- [ ] 间距 ≥6mil
- [ ] 过孔 ≥0.3mm
- [ ] 焊盘间距满足 SMT 要求

### 丝印
- [ ] 元件标号清晰
- [ ] 极性标识完整
- [ ] 连接器引脚 1 标识
- [ ] 板名/版本号丝印
