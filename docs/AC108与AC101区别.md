# AC108 与 AC101 区别

> X-Power（芯智汇）音频芯片，用于 ReSpeaker 6-Mic Circular Array 等麦克风阵列产品。

---

## 一、功能对比

| 项目 | AC108 | AC101 |
|------|-------|-------|
| **类型** | ADC（模数转换器） | DAC（数模转换器） |
| **信号方向** | 模拟 → 数字 | 数字 → 模拟 |
| **主要用途** | 采集麦克风信号 | 输出到耳机/扬声器 |
| **通道数** | 4 通道 | 立体声（2 通道） |
| **数字接口** | I2S / TDM 输出 | I2S 输入 |

---

## 二、在 ReSpeaker 6-Mic 中的角色

```
麦克风(模拟) ──→ AC108(×2) ──→ I2S/TDM ──→ 树莓派
树莓派 ──→ I2S ──→ AC101 ──→ 耳机/扬声器(模拟)
```

- **AC108 × 2**：两颗共 8 路 ADC，其中 6 路接麦克风，2 路为播放回采（echo）
- **AC101 × 1**：一颗立体声 DAC，驱动 3.5mm 耳机座和 JST 扬声器接口

---

## 三、简要总结

| 芯片 | 一句话说明 |
|------|-------------|
| **AC108** | 把麦克风模拟信号转为数字，送给树莓派 |
| **AC101** | 把树莓派数字音频转为模拟，送给耳机/喇叭 |
