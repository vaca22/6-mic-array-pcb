# SF32LB52 + 6 路麦克风设计指南

## 1. 方案概述

### 为什么选这个方案？

| 需求 | SF32LB52 能力 | 方案 |
|------|--------------|------|
| 6 路麦克风输入 | 仅 1 路 PDM 或 1 路模拟 ADC | 外置 ADAU1977 (8 通道 ADC) |
| 音频处理 | 内置 24 位 Codec | 直接使用内置 DAC 输出 |
| I2S 接口 | I2S1 完整支持 | 连接 ADAU1977 |
| 资源 | 16MB PSRAM + 16MB Flash | 足够音频缓冲和 AI 算法 |

### 信号流向
```
6x 模拟麦克风 → ADAU1977 (8 通道 ADC) → I2S → SF32LB52 → 内置 DAC → 功放 → 喇叭
                                        ↓
                                   AI 音频处理
                                   (小智/语音识别)
```

```
1 VDD  → 3.3V (MIC_BIAS)
2 GND  → GND
3 OUT  → ADAU1977 ADC 输入

```

| **WM8782** | 8 | 110dB | 216kHz | I2S | ~¥35 |

**推荐**: ADAU1977 (8 通道，立创有货，文档完善)

### 2.3 MCU - SF32LB52JUD6

| 参数 | 规格 |
|------|------|

| 存储 | 16MB NOR Flash (外置) |

| PDM | 1 路 PDM |
| 封装 | LGA-56 (6×6mm) |

**开发板**: SF32LB52-DevKit-Nano (R16N16 版本)

## 3. 原理图设计

### 3.1 麦克风输入 (6 路)

```
MIC1 OUT ────┬──── C1 ────┬──── ADAU1977 AIN1
             │            │
            R1           C2
             │            │
            GND          GND

MIC2 OUT ────┬──── C3 ────┬──── ADAU1977 AIN2
             │            │
            R2           C4
             │            │
            GND          GND

... (重复 6 路)
```

**元件值**:
- R1-R6: 1kΩ (偏置电阻，如麦克风需要)
- C1,C3,C5...: 1uF (耦合电容)
- C2,C4,C6...: 100nF (滤波电容)

### 3.2 ADAU1977 连接

```
ADAU1977 关键引脚:
┌────────────────────────────────────────┐
│                                        │
│  AIN1-AIN8  ← 6 路麦克风输入            │
│                                        │
│  AVDD 1-4   → 3.3V 模拟电源             │
│  DVDD 1-2   → 3.3V 数字电源             │
│  AGND 1-4   → 模拟地                    │
│  DGND 1-2   → 数字地                    │
│                                        │
│  MCLK      ← 12.288MHz (SF32 或晶振)   │
│  BCLK      → SF32LB52 PA29 (I2S_BCK)   │
│  LRCLK     → SF32LB52 PA30 (I2S_LRCK)  │
│  SDATA_OUT → SF32LB52 PA28 (I2S_SDI)   │
│                                        │
│  SCL       → SF32LB52 I2C_SCL (配置)   │
│  SDA       → SF32LB52 I2C_SDA (配置)   │
│                                        │
│  RST       → SF32LB52 GPIO (复位)      │
│                                        │
└────────────────────────────────────────┘
```

### 3.3 SF32LB52 连接

```
SF32LB52 (LCC 半孔引出):
┌────────────────────────────────────────┐
│                                        │
│  PA29 (I2S1_BCK)   ← ADAU1977 BCLK     │
│  PA30 (I2S1_LRCK)  ← ADAU1977 LRCLK    │
│  PA28 (I2S1_SDI)   ← ADAU1977 SDATA    │
│  PA24 (I2S1_MCLK)  → ADAU1977 MCLK     │
│                                        │
│  PA5 (MIC_BIAS)    → 麦克风供电        │
│  PA4 (MIC_ADC)     → 备用模拟输入      │
│                                        │
│  PA2 (DACP)        → 功放 IN+          │
│  PA3 (DACN)        → 功放 IN-          │
│                                        │
│  PA30 (GPIO)       → 功放使能          │
│                                        │
│  3.3V              → 数字电源          │
│  GND               → 地                │
│                                        │
│  USB-C             → 供电 + 调试       │
│                                        │
└────────────────────────────────────────┘
```

### 3.4 电源设计

```
USB 5V ────┬──── LDO (3.3V) ────┬──── 3.3V 数字
           │                    │
           │                   C10
           │                    │
           │                   GND
           │
           ├──── LDO (3.3V) ────┬──── 3.3V 模拟 (AVDD)
           │                    │
           │                   C11
           │                    │
           │                   GND
           │
           └──── 5V ────────────┴──── 功放电源 (如需要)
```

**LDO 推荐**:
- 数字电源：XC6206P332MR (3.3V, 150mA)
- 模拟电源：XC6206P332MR (独立供电，减少噪声)

## 4. PCB 布局

### 4.1 层叠设计 (4 层)

```
Layer 1: Signal (顶层 - 元件 + 关键信号)
Layer 2: GND (完整地平面)
Layer 3: PWR (电源平面)
Layer 4: Signal (底层 - 辅助走线)
```

### 4.2 布局要点

| 区域 | 要求 |
|------|------|
| 麦克风 | 间距 ≥15mm，声孔无铜，底层挖空 |
| ADAU1977 | 靠近麦克风组中心 |
| SF32LB52 | 靠近 ADAU1977，缩短 I2S 走线 |
| 电源 | 远离模拟信号，LDO 靠近负载 |
| USB-C | 板边，靠近 SF32LB52 |

### 4.3 走线规则

| 信号 | 线宽 | 要求 |
|------|------|------|
| I2S | 6mil | 等长，包地，<50mm |
| 模拟音频 | 6mil | 短而直，远离数字 |
| 电源 | ≥15mil | 尽量粗 |
| MCLK/BCLK | 6mil | 包地，无过孔 |

### 4.4 地平面

- 完整 GND 平面 (Layer 2)
- 模拟地 (AGND) 和数字地 (DGND) 单点连接
- 麦克风 GND 多点过孔到 GND 平面

## 5. 立创 EDA 操作步骤

### 5.1 创建项目
1. 打开 [立创 EDA 专业版](https://pro.lceda.cn)
2. 新建项目 → "6-Mic-SF32LB52"
3. 设置 4 层板

### 5.2 放置元件
1. 搜索 `SF32LB52` (如无比照官方封装画)
2. 搜索 `ADAU1977` (LQFP-48)
3. 搜索 `SPW0442` 或通用 MEMS 麦克风封装
4. 放置 USB-C 接口

### 5.3 原理图连接
按 3.1-3.4 节连接

### 5.4 PCB 布局
1. 先定位 6 个麦克风声孔
2. 放置 ADAU1977
3. 放置 SF32LB52
4. 电源和接口

### 5.5 布线
1. 先布 I2S 信号 (等长)
2. 布模拟音频线
3. 布电源线
4. 补全其他信号
