# 六路麦克风 PCB 设计指南

## 1. 为什么用音频 Hub？

### 传统方案 (无 Hub)
```
6x 麦克风 → 6 路 PDM 线 → MCU (需要 6 组引脚)
```
**问题**: MCU 引脚不够，走线复杂，时序难控制

### Hub 方案 (推荐)
```
6x 麦克风 → Hub → 1 路 I2S/TDM → MCU
```
**优势**: 
- 单路 I2S 输出 (3-4 根线)
- Hub 内部完成 ADC 和通道聚合
- MCU 资源占用最小

---

## 2. 音频 Hub 工作原理

### TDM (时分复用) 模式

```
I2S 时隙分配 (以 8 时隙为例):

| 时隙 0 | 时隙 1 | 时隙 2 | 时隙 3 | 时隙 4 | 时隙 5 | 时隙 6 | 时隙 7 |
|--------|--------|--------|--------|--------|--------|--------|--------|
|  MIC1  |  MIC2  |  MIC3  |  MIC4  |  MIC5  |  MIC6  |  空闲  |  空闲  |
```

**关键参数**:
- BCLK 频率 = 采样率 × 位深 × 时隙数
- 48kHz × 32bit × 8 = 12.288MHz

## 3. 元件选型详解

### 3.1 MEMS 麦克风

| 参数 | INMP441 | SPH0645LM4H |
|------|---------|-------------|
| 类型 | 数字 PDM | 数字 PDM |
| 灵敏度 | -38dBV | -38dBV |
| SNR | 65dB | 65dB |
| 供电 | 1.62-3.63V | 1.62-3.63V |
| 封装 | 4.72×3.76×1.25mm | 相同 |
| 价格 | ~¥3 | ~¥2.5 |

**L/R 引脚配置** (决定 PDM 时隙):
```
L/R 接 GND  → 左声道 (偶数时隙)
L/R 接 VDD  → 右声道 (奇数时隙)

6 路配置:
MIC1: L/R=GND  (时隙 0)
MIC2: L/R=VDD  (时隙 1)
MIC3: L/R=GND  (时隙 2)
MIC4: L/R=VDD  (时隙 3)
MIC5: L/R=GND  (时隙 4)
MIC6: L/R=VDD  (时隙 5)
```

### 3.2 音频 Hub 对比

| 芯片 | 通道数 | 接口 | 采样率 | 价格 | 推荐度 |
|------|--------|------|--------|------|--------|

| TLV320AIC3254 | 4 入 | I2S/TDM | 192kHz | ~¥30 | ⭐⭐⭐⭐ |

## 4. 立创 EDA 设计步骤

### 4.1 创建项目
1. 打开 [立创 EDA](https://lceda.cn)
2. 新建项目 → 命名 "6-Mic-Array"
3. 选择 **专业版** (推荐 4 层板)

### 4.2 放置元件
1. 元件库搜索型号 (如 "INMP441")
2. 放置 6 个麦克风，按阵列形状排列
3. 放置 ADAU1977 在中心位置
4. 放置 STM32F407 和电源部分

### 4.3 原理图连接
```
麦克风 → ADAU1977:
  VDD   → 3.3V
  GND   → GND
  PDM_OUT → ADAU1977 PDM 输入 (6 路)
  CLK   → ADAU1977 时钟输出
  L/R   → GND/VDD (时隙选择)

ADAU1977 → STM32:
  I2S_BCLK  → STM32 I2S_BCLK
  I2S_LRCLK → STM32 I2S_LRCLK  
  I2S_DOUT  → STM32 I2S_DIN
  MCLK      → STM32 MCLK (可选)
```

### 4.4 PCB 布局
1. **先定位麦克风孔**: 画 6 个 Keepout 圆 (直径 6-8mm)
2. **放置麦克风**: 围绕孔位，声孔对齐
3. **放置 Hub**: 靠近麦克风组中心
4. **放置 MCU**: 靠近 Hub，缩短 I2S 走线
5. **电源部分**: 远离模拟信号

### 4.5 布线规则
| 信号类型 | 线宽 | 间距 | 要求 |
|----------|------|------|------|
| PDM | 6mil | 6mil | 等长 ≤5mm |
| I2S | 6mil | 6mil | 包地，等长 |
| 电源 | 15mil | - | 尽量粗 |
| GND | - | - | 完整平面 |

## 5. 设计检查清单

### 原理图检查
- [ ] 所有电源引脚已连接
- [ ] 所有 GND 引脚已连接
- [ ] 去耦电容靠近芯片电源引脚
- [ ] 麦克风 L/R 引脚正确配置
- [ ] I2S 信号网络命名正确

### PCB 检查
- [ ] 麦克风声孔无铜皮遮挡
- [ ] 麦克风下方底层挖空
- [ ] PDM/I2S 走线等长
- [ ] 高速信号包地
- [ ] 完整地平面无断裂
- [ ] 电源走线足够宽
- [ ] 过孔不在麦克风附近

### DRC 检查
- [ ] 无短路错误
- [ ] 无开路错误
- [ ] 间距满足工艺要求
- [ ] 孔径满足工艺要求

## 6. 下单指南

### 嘉立创 PCB 参数
```
层数：4 层
板厚：1.6mm
板材：FR-4
铜厚：1oz
阻焊：绿色 (或其他)
字符：白色
表面处理：沉金 (ENIG) - 推荐音频应用
最小线宽/距：6mil
最小孔径：0.3mm
```

### BOM 配单
1. 立创 EDA → BOM 工具 → 一键配单
2. 确认所有元件有库存
3. 选择 SMT 贴装 (推荐)

## 7. 调试建议

### 上电前检查
- [ ] 万用表测 3.3V 对地阻抗 (>100Ω)
- [ ] 检查电源无短路

### 上电后测试
1. 测各点电压 (3.3V, 1.8V 等)
2. 用示波器看 MCLK/BCLK 时钟
3. 用逻辑分析仪抓 I2S 波形
4. 软件读取音频数据验证

### 常见问题
| 问题 | 可能原因 | 解决 |
|------|----------|------|
| 无声音 | 时钟缺失 | 检查晶振/MCU 配置 |
| 噪声大 | 地平面不完整 | 检查 GND 连接 |
| 串音 | 麦克风隔离不够 | 增加间距/挖空 |
| 时隙错乱 | L/R 配置错误 | 检查接线 |
